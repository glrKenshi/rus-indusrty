# РУС-Индустрия

Сайт компании РУС-Индустрия — оборудование и отечественное ПО для цифровых пищевых производств.

## Технологии

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:8080

## Сборка

```bash
npm run build
```

## Другие команды

- `npm run preview` — предпросмотр production-сборки
- `npm run lint` — проверка кода ESLint
- `npm run format` — форматирование Prettier
- `npm run test:unit` — юнит-тесты
- `npm run test:e2e` — E2E тесты (Playwright)
